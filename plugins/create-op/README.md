# create-op Plugin

User-defined plugin for `create-op` with Codex-compatible execution.

## What you can customize

- `skills/create-op/references/user-config.md`

- `tt_metal/third_party/tt_ops_code_gen/scripts/start_ttnn_role_sequence.sh`
