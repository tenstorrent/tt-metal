Return value: uint64_t

Argument

Description

Data type

Valid range

required

Physical x coordinate of core

uint32_t

True

uint32_t

WH: 0-11, BH: 0-11

True

addr

uint32_t

0..1MB

True

noc

uint8_t

0 or 1

False
