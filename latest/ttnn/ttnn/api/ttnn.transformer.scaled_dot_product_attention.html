Causal scaled dot product attention. This API mimicks the PyTorch API of the same name. The implementation is FlashAttention-2.‚Äù

Accepts a SDPAProgramConfig which specifies the grid size and chunk tiles in the Q and K sequence lengths. The op parallelizes over b, nqh, and Q‚Äôs s dimension.
