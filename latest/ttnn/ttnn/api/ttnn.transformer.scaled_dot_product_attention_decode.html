A version of scaled dot product attention specifically for decode. The implementation is Flash-Decode and it currently only supports MQA on decoding single token.

Accepts a SDPAMultiCoreProgramConfig which specifies the grid size and chunk tiles in the K/V/Mask sequence lengths (Q chunk tiles is not used). The op parallelizes over b and K/V/Mask‚Äôs s dimension.
