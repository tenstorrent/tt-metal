# FIBO 4×8 Galaxy bring-up (SP=4, TP=8) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Run the Bria FIBO text→image pipeline correctly on a 4×8 Blackhole Galaxy mesh with SP=4 (axis 0) and TP=8 (axis 1).

**Architecture:** FIBO's parallel config is already shape-driven, so a `(4,8)` mesh auto-yields SP=4/TP=8. The one hard blocker is the per-block text injection (`inject_text`), hard-coded to TP==2. We replace it with a TP-general all-gather → concat-halves → reshard path and drop the TP==2 mask, then validate on hardware via the transformer mesh unit test and the pipeline PCC test.

**Tech Stack:** Python, ttnn, pytest, tt-metal `models/tt_dit` DiT infra (`CCLManager`, `DiTParallelConfig`, `ColParallelLinear`, ring SDPA).

## Global Constraints

- Functional-first: correctness only. No matmul/SDPA/encoder-FSDP/VAE-blocking tuning in this plan (documented follow-ups in the spec).
- Mesh convention: `sp_axis=0`, `tp_axis=1`, `cfg=(1,0)`. Factors derived from `mesh_device.shape`.
- Hardware: validate on the physical 4×8 BH Galaxy (available). `ARCH_NAME=blackhole`.
- Do NOT guess `num_links`/topology — pin empirically on hardware.
- Reference run env for FIBO tests: `HF_HUB_OFFLINE=1 TT_METAL_HOME=$PWD PYTHONPATH=$PWD ARCH_NAME=blackhole python_env/bin/python -m pytest <test> -v -s`. FIBO weights must be cached (`FIBO_PATH`, default `briaai/FIBO`); tests skip if not.
- Spec: `docs/superpowers/specs/2026-07-14-fibo-4x8-mesh-design.md`.

- [ ] **Step 3: Replace `_InjectionMask` + `inject_text` with the TP-general path**

In `transformer_bria_fibo.py`, delete the entire `_InjectionMask` class (lines 108-153) and replace `inject_text` (lines 156-204) with:

```python
def inject_text(
    encoder_hidden_states: ttnn.Tensor,
    projected: ttnn.Tensor,
    *,
    ccl_manager: CCLManager | None = None,
    tp_axis: int | None = None,
) -> ttnn.Tensor:
    """Concat-halves text injection for FIBO blocks, general over the TP factor.

    Replaces the upper half of the context (prompt) features with the per-block projected
    text embedding::

        out[..., :half] = encoder_hidden_states[..., :half]
        out[..., half:] = projected

    where ``half = inner_dim // 2``. ``projected`` is the full ``half`` on every device
    (it comes from a *replicated* ``BriaFiboTextProjection`` linear).

    Two paths:

    * **tp=1** (``ccl_manager`` / ``tp_axis`` is ``None``): ``encoder_hidden_states`` is the full
      ``[batch, P, inner_dim]`` context; plain feature-dim concat of its first half with ``projected``.
    * **tp>1**: ``encoder_hidden_states`` is the local feature shard ``[batch, P, inner_dim / tp]``
      on ``tp_axis``. All-gather it to the full ``inner_dim`` (replicated across ``tp_axis``), do the
      concat-halves in full space, then ``mesh_partition`` back to the local shard on ``tp_axis``.
      The gather makes this correct for any TP factor (the tp==2 shard-aligned special case is gone).

    Args:
        encoder_hidden_states: tp=1: ``[batch, P, inner_dim]``; tp>1: local shard ``[batch, P, inner_dim/tp]``.
        projected: per-block projection ``[batch, P, inner_dim // 2]`` (replicated on tp).
        ccl_manager: CCL manager for the tp>1 all-gather, or ``None`` for tp=1.
        tp_axis: tensor-parallel mesh axis for the tp>1 all-gather/reshard, or ``None`` for tp=1.

    Returns:
        Injected context, same feature layout as ``encoder_hidden_states``.
    """
    half = projected.shape[-1]

    if ccl_manager is None or tp_axis is None:
        assert (
            encoder_hidden_states.shape[-1] == 2 * half
        ), f"inject_text: expected encoder_hidden_states last dim {2 * half}, got {encoder_hidden_states.shape[-1]}"
        return ttnn.concat([encoder_hidden_states[:, :, :half], projected], dim=-1)

    # tp>1: gather the local feature shard to the full inner_dim (replicated across tp_axis),
    # concat-halves in full space against the replicated projection, then reshard on tp_axis.
    enc_full = ccl_manager.all_gather(encoder_hidden_states, dim=-1, mesh_axis=tp_axis, use_hyperparams=True)
    injected_full = ttnn.concat([enc_full[:, :, :half], projected], dim=-1)
    return ttnn.mesh_partition(injected_full, dim=-1, cluster_axis=tp_axis)
```
