# Yolov11n

## Platforms:
    Wormhole (n150, n300), Blackhole (p150)

## Introduction
**YOLOv11n** is the smallest variant in the YOLOV11 series, it offers improvements in accuracy, speed, and efficiency for real-time object detection. It features enhanced architecture and optimized training methods, suitable for various computer vision tasks.

## Prerequisites
- Cloned [tt-metal repository](https://github.com/tenstorrent/tt-metal) for source code
- Installed: [TT-Metalium™ / TT-NN™](https://github.com/tenstorrent/tt-metal/blob/main/INSTALLING.md)

## How to Run

Find yolov11n instructions for the following devices:

- Wormhole (n150, n300): [models/demos/wormhole/yolov11](https://github.com/tenstorrent/tt-metal/tree/main/models/demos/wormhole/yolov11)

- Blackhole (p150): [models/demos/blackhole/yolov11](https://github.com/tenstorrent/tt-metal/tree/main/models/demos/blackhole/yolov11)
