# Yolov8x

## Platforms:
Wormhole (n150, n300), Blackhole (p150)

## Introduction
YOLOv8 is one of the recent iterations in the YOLO series of real-time object detectors, offering cutting-edge performance in terms of accuracy and speed.

## Prerequisites
- Cloned [tt-metal repository](https://github.com/tenstorrent/tt-metal) for source code
- Installed: [TT-Metalium™ / TT-NN™](https://github.com/tenstorrent/tt-metal/blob/main/INSTALLING.md)

## How to Run
Find yolov8x instructions for the following devices:

- Wormhole (n150, n300): [models/demos/wormhole/yolov8x](https://github.com/tenstorrent/tt-metal/tree/main/models/demos/wormhole/yolov8x)

- Blackhole (p150): [models/demos/blackhole/yolov8x](https://github.com/tenstorrent/tt-metal/tree/main/models/demos/blackhole/yolov8x)
