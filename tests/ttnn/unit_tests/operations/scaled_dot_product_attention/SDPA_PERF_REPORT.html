Final case-4 setup (h8 · S16384 · d512 · 60% HiFi2)

Work unit = (head, 4-tile-row Q-chunk); 1024 units, 16/core, all 64 cores. Per head, g=8 cores share the KV stream. c_q=4, c_kv=4, Dt=16, Nkv=128 blocks/unit. Compute config: HiFi2, bf16 DEST, half-sync.

Reader (NoC0)
Q chunk ⇒ pre-scaled by compute · Kᵀ: diagonal leader reads 266 KB, row-mcasts; sems 0/1
Writer (NoC1)
V: leader reads 266 KB, mcasts; sems 2/3 · drains output chunk
Compute per KV block
QKᵀ (1×4 sub) + exp on DEST → probs · rowsum L1-acc · P·V 2×4 pinned L1-acc
Unit end
recip(l)+Newton · O′·inv (subblock-major offset chains) → out
