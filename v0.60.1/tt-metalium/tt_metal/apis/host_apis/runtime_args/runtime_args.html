void tt::tt_metal::SetRuntimeArgs(const Program &program, KernelHandle kernel, const std::variant<CoreCoord, CoreRange, CoreRangeSet> &core_spec, stl::Span<const uint32_t> runtime_args)

Set runtime args for a kernel that are sent to the core during runtime. This API needs to be called to update the runtime args for the kernel. Maximum of 255 allowed runtime args per core (unique and common runtime args count toward same limit).

Return value: void

Argument

Description

Type

Valid Range

Required

Yes

Location of Tensix core(s) where the runtime args will be written
