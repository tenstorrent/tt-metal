### 92e3edfa4cc — FIX PG (#42429): skip FIX AY when FIX AC MMIO heartbeat poll fails

**Type**: PRODUCTION code fix to `risc_firmware_initializer.cpp`. Two sub-fixes:

#### Sub-fix 1: FIX PG — skip FIX AY when ac_heartbeat_any_ready=false

Root cause:
- FIX AC reboots MMIO ETH channels during teardown, then polls for heartbeat
  incrementing to confirm firmware is back at base-UMD.
- If ALL MMIO ETH heartbeats time out (`ac_heartbeat_any_ready` stays false),
  the relay is NOT restored — but FIX AY would still run.
- FIX AY performs deferred non-MMIO ERISC reset via the restored MMIO relay.
  With no relay restoration, every FIX AY attempt polls 5s per channel per
  non-MMIO device (4 channels × 4 non-MMIO devices = up to 80s wasted).

FIX PG fix:
- Declares `bool ac_heartbeat_any_ready = false` outside the heartbeat block.
- Sets `ac_heartbeat_any_ready = true` when any poll_state is ready.
- FIX AY entrance guarded: `if (get_control_plane_ && ac_heartbeat_any_ready)`
- If not ac_heartbeat_any_ready: log warning + skip FIX AY, note that FIX BC
  handles non-MMIO on next SetUp.

#### Sub-fix 2: FIX SB (GAP-76) — guard IDLE_ETH not_done_cores.insert

Root cause (found in the same commit):
- `initialize_and_launch_firmware()` loops over `eth_cores_to_init` (IDLE_ETH cores).
- `initialize_firmware(HalProgrammableCoreType::IDLE_ETH, ...)` returns early
  (`break`) when `INIT_FABRIC` is not in FabricManagerMode (line ~2156).
- Before FIX SB: `not_done_cores.insert(virtual_core)` called unconditionally
  after the early-return path.
- `deassert_risc_reset_at_core` fires for the uninitialized core.
- Core starts executing stale L1 firmware that writes 0x55 to run_mailbox.
- `wait_until_cores_done` TT_FATALs → SIGABRT (exit 134).

FIX SB fix:
```cpp
if (has_flag(descriptor_->fabric_manager(), tt_fabric::FabricManagerMode::INIT_FABRIC)) {
    not_done_cores.insert(virtual_core);
}
```

**Decision: HIGH-PRIORITY — GAP-76 implemented.**

This is a production code change with two regression paths:
1. FIX PG: prevents N×5s wasted polls in teardown when MMIO heartbeats all fail.
2. FIX SB: prevents SIGABRT (TT_FATAL) in wait_until_cores_done when IDLE_ETH
   cores exist but INIT_FABRIC is absent from FabricManagerMode.
