# tt-umd scripts

## Clone the tt-umd repository

```
git clone https://github.com/tenstorrent/tt-umd.git

cd tt-umd

```

Copy scripts into the tt-umd repo

## Installing dependencies

```
./install-dependencies.sh
```

## Build tt-umd for wormhole

```
./build-umd.sh wormhole_b0
```

## Build tt-umd for blackhole

```
./build-umd.sh blackhole
```

## Run tt-umd tests for wormhole

```
./tests-umd.sh wormhole_b0
```

## Run tt-umd tests for blackhole

```
./tests-umd.sh blackhole
```
