Calls `fabric_unicast_noc_scatter_write(&sender, packet_header, dst_dev_id, dst_mesh_id, src_l1_addr, scatter_size, NocUnicastScatterCommandHeader{dst_noc_addr0, dst_noc_addr1})` where `dst_noc_addr0 = safe_get_noc_addr(rx_noc_x, rx_noc_y, dst_base_addr, 0)` and `dst_noc_addr1 = safe_get_noc_addr(rx_noc_x, rx_noc_y, dst_base_addr + scatter_offset, 0)`. After data send, sends separate atomic_inc for completion. Scatter payloads MUST be <= MAX_PACKET_SIZE (passthrough, no chunking).

2. **fused_atomic_inc_unicast_tx_writer_raw.cpp:** Same RT args as unicast_tx_writer_raw.cpp. Calls `fabric_unicast_noc_fused_unicast_with_atomic_inc(&sender, packet_header, dst_dev_id, dst_mesh_id, src_l1_addr, total_size, NocUnicastAtomicIncFusedCommandHeader{dst_noc_addr, sem_noc_addr, 1, 32})`. Does NOT send separate atomic_inc after -- the fused wrapper fires it on the final chunk. Sender close after call.

3. **fused_scatter_atomic_inc_unicast_tx_writer_raw.cpp:** Same RT args as scatter kernel plus scatter_offset. Calls `fabric_unicast_noc_fused_scatter_write_atomic_inc(&sender, packet_header, dst_dev_id, dst_mesh_id, src_l1_addr, scatter_size, NocUnicastScatterAtomicIncFusedCommandHeader{dst_noc_addr0, dst_noc_addr1, sem_noc_addr, 1, 32})`. No separate atomic_inc. Small payload only (passthrough).

All kernels: include `mesh/api.h` and `linear/api.h`, use `namespace tt::tt_fabric::mesh::experimental`, build sender from args, open/close sender.
