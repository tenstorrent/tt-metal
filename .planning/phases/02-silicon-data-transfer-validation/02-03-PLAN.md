   - Rebuild and re-run
4. Repeat until ALL tests pass

IMPORTANT: Device tests run one at a time (serialized). Do not attempt parallel test execution.
IMPORTANT: If a test hangs for > 60 seconds, it likely means the receiver semaphore is never being incremented. Check completion signaling logic.
