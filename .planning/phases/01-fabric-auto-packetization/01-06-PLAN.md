**Function 1: fabric_multicast_noc_fused_scatter_write_atomic_inc (addrgen, single-sender)**

Pattern from lines 4149-4206 with these changes:
- Template: `<typename FabricSenderType, typename AddrGenType>`
- Additional params: `page_id0`, `page_id1`, `semaphore_noc_address`, `val`, `offset0 = 0`, `offset1 = 0`, `flush = true`
- Compute `page_size = addrgen_detail::get_page_size(addrgen)` and `noc_addr0 = addrgen_detail::get_noc_address(addrgen, page_id0, offset0)`, `noc_addr1 = addrgen_detail::get_noc_address(addrgen, page_id1, offset1)`.
- Check: `if (page_size * 2 <= FABRIC_MAX_PACKET_SIZE)` — single scatter packet path (no chunking).
- Else (large page): fall back to two separate unicast writes per chunk pair (each chunk sends `FABRIC_MAX_PACKET_SIZE` bytes to each scatter target independently), then fused scatter on final chunk.
  - For intermediate chunks: call `fabric_multicast_noc_unicast_write_single_packet<FabricSenderType, false>` twice (once for each scatter target's noc_addr), advancing both `noc_addr0 += FABRIC_MAX_PACKET_SIZE` and `noc_addr1 += FABRIC_MAX_PACKET_SIZE` per chunk.
  - For final chunk: call `fabric_multicast_noc_fused_scatter_write_atomic_inc_single_packet<FabricSenderType, false>` with `NocUnicastScatterAtomicIncFusedCommandHeader{noc_addr0, noc_addr1, semaphore_noc_address, val, flush}`.
  - Set route once before loop: `fabric_set_mcast_route(packet_header, dst_dev_id, dst_mesh_id, ranges.e, ranges.w, ranges.n, ranges.s)`.
  - Call `noc_async_writes_flushed()` before each chunk (including first).
  - `src_addr` advances by `FABRIC_MAX_PACKET_SIZE` per chunk.

**Function 2: fabric_multicast_noc_fused_scatter_write_atomic_inc (addrgen, conn mgr)**
