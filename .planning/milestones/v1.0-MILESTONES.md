## v1.0 — Fabric Auto-Packetization

**Shipped:** 2026-03-11
**Phases:** 1 | **Plans:** 8 | **Commits:** 27

### Delivered

Transparent auto-packetization for all fabric write APIs. Payloads larger than FABRIC_MAX_PACKET_SIZE are automatically chunked with correct NOC address advancement, breadth-first multi-connection ordering, and fused-op final-chunk-only atomic_inc semantics.

### Key Accomplishments

1. 9 raw-size write API families renamed to `_single_packet` in both linear/api.h and mesh/api.h (18 function declarations)
2. 18 auto-packetizing wrappers under original names with chunking loops
3. Breadth-first multi-connection ordering for connection manager variants
4. 6 new addrgen overloads for `multicast_fused_scatter_write_atomic_inc` in mesh/api.h
5. Device toolchain compile tests (1D + 2D) for all 9 wrapper families
6. Zero caller changes required — existing code transparently gets auto-packetization
