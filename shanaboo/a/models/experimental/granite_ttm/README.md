# Granite Timeseries TTM-R1 (Tiny Time Mixer) on TT-Metal

This directory contains the TT-Metal implementation of **IBM Granite Timeseries TTM-R1** (Tiny Time Mixer) using TTNN APIs.

## Overview

Granite TTM-R1 is a revolutionary compact pre-trained foundation model for multivariate time-series forecasting with **< 1 million parameters**. It achieves state-of-the-art performance rivaling models with billions of parameters in zero-shot and few-shot scenarios.

## Key Features

- **Ultra-lightweight**: < 1M parameters, < 5MB model size
- **Fast inference**: < 5ms latency, 2000+ sequences/second
- **Zero-shot & few-shot**: Works out-of-the-box or fine-tune with < 5% data
- **Optimized for**: Context length 512, forecast length 96
- **Multi-resolution**: 10min, 15min, 1-hour forecasting
- **Edge deployment**: Optimized for resource-constrained environments
