ttnn.lerp_bw = Operation(python_fully_qualified_name='ttnn.lerp_bw', function=<ttnn._ttnn.operations.ternary_backward.lerp_bw_t object>, preprocess_golden_function_inputs=<function default_preprocess_golden_function_inputs>, golden_function=<function _golden_function>, postprocess_golden_function_outputs=<function default_postprocess_golden_function_outputs>, is_cpp_operation=True, is_experimental=False)ÔÉÅ

Performs backward operations for lerp of input_tensor_a, input_tensor_b and input_tensor_c or scalar with given grad_tensor.

Parameters:

  • grad_tensor (ttnn.Tensor) ‚Äì the input gradient tensor.

  • input_tensor_a (ttnn.Tensor) ‚Äì the input tensor.

  • input_tensor_b (ttnn.Tensor) ‚Äì the input tensor.

  • input_tensor_c (ttnn.Tensor or Number) ‚Äì the input tensor.

Keyword Arguments:

memory_config (ttnn.MemoryConfig, optional) ‚Äì memory configuration for the operation. Defaults to None.

Returns:

List of ttnn.Tensor ‚Äì the output tensor.

Note

Supported dtypes, layouts, and ranks:

For Inputs : input_tensor_a, input_tensor_b and input_tensor_c

Dtypes

Layouts

Ranks

BFLOAT16

TILE

2, 3, 4

For Inputs : input_tensor_a, input_tensor_b and scalar

Dtypes

Layouts

Ranks

BFLOAT16, BFLOAT8_B

TILE

2, 3, 4

bfloat8_b/bfloat4_b is only supported on TILE_LAYOUT

Example

>>> grad_tensor = ttnn.from_torch(torch.tensor([[1, 2], [3, 4]], dtype=torch.bfloat16), layout=ttnn.TILE_LAYOUT, device=device)
>>> tensor1 = ttnn.from_torch(torch.tensor([[1, 2], [3, 4]], dtype=torch.bfloat16, requires_grad=True), layout=ttnn.TILE_LAYOUT, device=device)
>>> tensor2 = ttnn.from_torch(torch.tensor([[1, 2], [3, 4]], dtype=torch.bfloat16, requires_grad=True), layout=ttnn.TILE_LAYOUT, device=device)
>>> tensor3 = ttnn.from_torch(torch.tensor([[1, 2], [3, 4]], dtype=torch.bfloat16, requires_grad=True), layout=ttnn.TILE_LAYOUT, device=device)
>>> output = ttnn.lerp_bw(grad_tensor, tensor1, tensor2, tensor3/scalar)
