# golden: layer_norm_rm
/create-op
Create a TTNN operation called layer_norm_rm that performs layer normalization on
row-major interleaved tensors.

The operation normalizes each row of the input tensor through the following steps:
1. Compute the row-wise mean
2. Subtract the mean from each element to get centered values
3. Square the centered values and compute their mean to obtain the variance
4. Add epsilon (a configurable operation parameter) to the variance for numerical stability
5. Compute the reciprocal square root of (variance + epsilon)
6. Multiply centered values by this reciprocal square root to produce standardized output

After standardization, apply learnable affine parameters gamma (scale) and
beta (shift). Both gamma and beta are row-major tensors with shape (1, 1, 1, W)
where W matches the input width.

The input tensor must be at least 2D, and the output retains identical dimensions.
Input dtype is bfloat16 with the last two dimensions aligned to tile size (32),
but the implementation should be dtype-agnostic where possible.

Execute in FULLY AUTOMATED mode. Make reasonable assumptions autonomously and
DO NOT prompt for user confirmation or clarification at any point.

Enable FULL BREADCRUMBS LOGGING throughout execution.
