Six schemes — WC-NC swing
============================================================================================================================================================================================================================

ALL-STOP  : every thread does perf MMIO
PER-CFG   : compile-gate, one thread only; others empty (asymmetric → L1_TO_L1 broken)
ATINCGET  : runtime gate via TT_ATINCGET; broke MATH_ISO via THCON-FPU interference
TENSIX_S  : all-stop + tensix_sync barriers; clean (= all-stop)
SPIN_L1   : one active + spinwait poll on L1 sync_addr; UNPACK_ISO regression from L1 contention
PCBUF_SEM : one active + pc_buf semaphore barrier (NEW); per-TRISC pc_buf MMIO, no L1 fabric contention
