========================================================================================================================================================================================================
Five schemes — WC-NC swing (smaller |Δ| better)
========================================================================================================================================================================================================

ALL-STOP : every thread arm/freeze/read unconditionally
PER-CFG  : compile-time gate, only one thread does perf work; others empty
ATINCGET : runtime gate via TT_ATINCGET, first-arms/last-freezes; others pass through
TENSIX_S : all-stop + tensix_sync() barrier in ctor/dtor
SPINWAIT : one thread (per run_type) does freeze+read; others spinwait on SYNC flag
           — keeps symmetric delay AND only one thread saturates debug-reg bus
